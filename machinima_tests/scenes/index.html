Machinima Test Scenes

Basic Example with super-hands

Create html files in this directory for the scenes you want to use in machinima testing. Don't use script tags in the head or body tags because they won't be included with the a-scene when it is injected into the test enviroment. Instead, include any external components/dependencies by require-ing them in machinima_tests/main.js.

Once in the scene, press space to start and end recording. Save the downloaded json file to machinima_tests/recordings and load it in the corresponding test script.
